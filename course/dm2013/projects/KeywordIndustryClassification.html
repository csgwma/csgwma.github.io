Final Project： Keyword industry classification(关键词行业分类）

Dataset Description

The dataset is derived from Baidu’s industry classification system,including about one million keywords and other relevant information.The details are in the following table. dataset download

Field format remarks
keyword string content
class_ID string from 1 to 33,each number means one category
advertiser string advertisers’ id,which buy this keyword,is separated by commas

Task

Now,you need design a classification algorithm or use any existing methods ,such as SVM,CNN and etc ,to learn a classifier for this dataset.And you can incorporate some or all of these offered information to your algorithm.The most important part is that you must submit a report finally,which includes data preprocessing,algorithm and experiments.You had better utilize various criteria to show the classification effect.
